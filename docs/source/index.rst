Introduction
============
:mod:`SPARCS` is a `snakemake <https://snakemake.readthedocs.io/en/stable/>`_ pipeline for the transcriptome-wide
prediction of riboSNitches.

The SPARCS pipeline makes it easy to predict the effects of SNPs on local RNA secondary structure. Built entirely in
`snakemake <https://snakemake.readthedocs.io/en/stable/>`_, SPARCS is designed to be modular and easy to customize.

Features
--------
- Modular design allowing for easy customization
- Automatic dependency management
- Ability to utilize custom gene models
